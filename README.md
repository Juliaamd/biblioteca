📚 Sistema de Biblioteca — UNIALFA

Este projeto foi desenvolvido como um protótipo de sistema de gerenciamento de biblioteca, com foco em organização de acervo, usabilidade e visual moderno.

Escolhemos desenvolver um sistema de biblioteca pois ele permite aplicar conceitos importantes de desenvolvimento, como manipulação de dados, interface interativa, organização de informações e simulação de um sistema real.

Além disso, é um sistema que gostríamos de ter na faculdade para fazer melhor uso do acervo disponível em nossa biblioteca.

## Funcionalidades iniciais

- Listagem de livros
- Adição de novos livros
- Alterar status (Disponível / Emprestado)
- Remoção de livros
- Busca por título ou autor
- Estatísticas (total, disponíveis, emprestados)
- Paginação de livros
